# История изменений

## v1.1.0-dev.3

- Добавлено новое rust-ядро.
- Добавлен выбор ядра клиента: go / rust в настройках производительности.
- В логах теперь явно видно активное ядро.
- Расширена статистика по траффику.
- Добавлен packet pool и оптимизации dispatcher path в go-ядро.
- Реализован экспериментальный rust data path: UDP/TCP TURN + DTLS.
- Rust принимает cli флаги, что и go-ядро.
- Добавлена поддержка split-tunnel rewrite конфига.
- Добавлены retry/backoff, credential TTL rotation, ранняя ротация при quota/session death.
- Добавлено debug меню (пока для галочки, функционал скудный).
- Диагностика в главом меню верно опоеделяет доступность интернета с включенным туннелем.
- В главных виджетах значения теперь анимируются посимвольно, без дергания всего блока.
- Раздел «Роутинг» очищен от лишних подпунктов/текстов.
- В роутинге добавлены плавные анимации цвета, текста, бейджа, раскрытия параметров и строк приложений.
- Редактирование ноды теперь открывается зажатием по ноде.
- В редакторе ноды пароль скрывается.
- Индикатор проверки обновлений стал появляться плавно.
- Прогресс скачивания APK теперь плавный, с wave анимацией.

## v1.1.0-dev.2

- Многочисленные фиксы проверки обновлений: зависшая кнопка, баги UI, таймауты.
- В обновлениях добавлены «Доступные обновления».
- Удален DataSync для фикса крашей Android 14+.
- Фикс всех элементов, проигрывающих Enter-анимацию без взаимодействия пользователя.
- «Позже» больше не убирает обновление из списка доступных обновлений.
- Changelog в разделе обновлений снова показывает ветку текущего установленного APK.
- Диалог «Доступно обновление» больше не открывается сам при входе в раздел «Обновления» или после ручной проверки.
- Ошибки проверки обновлений больше не сбрасывают таймер проверки в бесконечные повторы раз в минуту.

## v1.1.0-dev.1

- Добавлен preview импортируемой ноды перед сохранением.
- Добавлены настройки производительности: потоки, Wi-Fi high performance, keepalive.
- Исправлено ручное выключение VPN без ложного сообщения об аварийном обрыве.
- Улучшена устойчивость запуска WireGuard.
