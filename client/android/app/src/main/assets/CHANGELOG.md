# История изменений

## v1.1.0-dev.2

- Многочисленные фиксы проверки обновлений: зависшая кнопка, баги UI, таймауты.
- В обновлениях добавлены «Доступные обновления».
- Удален DataSync для фикса крашей Android 14+.
- Фикс всех элементов, проигрывающих Enter-анимацию без взаимодействия пользователя.
- «Позже» больше не убирает обновление из списка доступных обновлений.
- Changelog в разделе обновлений снова показывает ветку текущего установленного APK.
- Диалог «Доступно обновление» больше не открывается сам при входе в раздел «Обновления» или после ручной проверки.
- Ошибки проверки обновлений больше не сбрасывают таймер проверки в бесконечные повторы раз в минуту.

## v1.1.0-dev.1

- Добавлен preview импортируемой ноды перед сохранением.
- Добавлены настройки производительности: потоки, Wi-Fi high performance, keepalive.
- Исправлено ручное выключение VPN без ложного сообщения об аварийном обрыве.
- Улучшена устойчивость запуска WireGuard.
